This Yelp App uses the Yelp restaurant API to find restaurants around the 55416 (Minneapolis) area.

To build the app, open up the ‘Yelp.xcodeproj’ in Xcode, and build to an ios device/simulator.

To use the app, click on the search bar and enter your search term. For example, ‘pizza’ will show pizza restaurants in the area.

Click on a specific restaurant result to view the name, rating, and phone number for a particular restaurant. 

Navigate back to search for a new term or pick a different entry.

Unit tests are available under the ‘Yelp App Test’ directory in Xcode.
